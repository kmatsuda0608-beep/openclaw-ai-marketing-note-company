# Learning Roadmap: AI & Web Marketing v1

Last updated: 2026-04-16
Owner: Ciel
Brand: `nanda AI`

## 1. 結論
可能。
ただし、単に情報を大量に読むより、**学習 → 要約 → 実務化 → 投稿化 → 振り返り** のループで進める。

目指すのは、
- AIの最新動向を追える
- Webマーケの基本と実務を説明できる
- 収集した情報を、日本向けの投稿や案件提案に変換できる
- おまつさんの実務支援に使える
状態。

## 2. 学習対象の大枠

### A. AI領域
- モデル / ツールの最新動向
- AIエージェント
- ワークフロー自動化
- コンテンツ制作へのAI活用
- 業務効率化ユースケース
- 画像 / 動画 / 音声系AI

### B. Webマーケ領域
- 広告運用
- SEO
- SNS運用
- コンテンツマーケティング
- 導線設計
- CVR改善
- オファー設計

### C. つなぎ目
- AIをWebマーケ実務へどう入れるか
- SNSを案件獲得や商品導線へどう接続するか
- 仕組み化して継続運用できる形にどう落とすか

## 3. 学び方の原則

1. 海外ソースを先に拾う
2. 日本の実務者にどう効くかへ翻訳する
3. 実務で使える粒度まで落とす
4. 投稿・運用・提案に再利用する
5. 学んだだけで終わらせず、資産化する

## 4. 学習フェーズ

### Phase 1, 基礎の地図を作る
- AIの主要プレイヤー、主要カテゴリを把握
- Webマーケの基本構造を整理
- 用語の取り違えを減らす

### Phase 2, 実務理解を深める
- 広告、SEO、SNS、導線設計を実務目線で理解
- AIツールを実務に入れる観点を整理
- 海外事例の再解釈精度を上げる

### Phase 3, 発信に変換する
- X / Threads向け構成へ落とす
- 実務者向けに意味づけして投稿化
- 保存される投稿と案件導線を両立する

### Phase 4, 専門性を強くする
- 伸びたテーマを深掘り
- よく使う論点をテンプレ化
- note、商品、提案資料へ展開

## 5. 毎日の学習サイクル

### 毎日
- 海外ソースを確認
- 重要トピックを3〜5件抽出
- そのうち1〜2件を深掘り
- 日本向けの意味づけを作る
- 投稿候補かナレッジに変換する

### 毎週
- 反応が良かったテーマを分析
- 理解が浅い分野を補強
- よく出る話題の背景知識を追加学習

## 6. 学習の成果物

- ソースウォッチリスト
- 日次リサーチメモ
- 投稿候補
- 実務で使う判断メモ
- noteや商品に転用できるナレッジ

## 7. ゴールの定義

短期ゴール:
- 海外AI / Webマーケ情報を毎日拾って整理できる
- 投稿案へ安定変換できる

中期ゴール:
- AIとWebマーケを横断して解説できる
- 実務者向けの示唆を継続的に出せる

長期ゴール:
- nanda AI を、AI × Webマーケの実務翻訳者 / 設計者として確立する
- 案件、商品、コミュニティ導線までつなげる
