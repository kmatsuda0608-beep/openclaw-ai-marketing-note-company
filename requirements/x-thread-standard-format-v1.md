# X Thread Standard Format v1

Last updated: 2026-04-16
Brand: `nanda AI`

## 1. 目的

Xスレッドを量産しても品質が崩れにくいよう、基本フォーマットを固定する。

## 2. 基本構成

### 1ポスト目
- 強い結論 or 問題提起
- 読む理由を1文で提示
- 長くしすぎない

### 2〜4ポスト目
- 背景
- よくある失敗
- なぜそうなるか

### 5〜7ポスト目
- 解決の考え方
- 実務での整理法
- AIをどう使うか

### 8ポスト目以降
- まとめ
- 次の行動
- CTA

## 3. 推奨本数
- 5〜8ポストを基本
- 重いテーマでも最大10ポストまで

## 4. トーン
- 実務的
- 断定しすぎない
- 煽りすぎない
- すぐ使える感を出す

## 5. CTAルール
- 初見には弱め
- 比較検討層には中くらい
- 強CTAは連発しない

## 6. 禁止事項
- 未確認の売上実績
- 誰でも必ずできる系の断定
- 過剰な不安煽り
- AI万能論

## 7. 運用メモ
- 1スレッド1テーマ
- 1ポスト1メッセージに寄せる
- 投稿後はシートへURLと数値を戻す
