# Threads Writing Structure v1

Last updated: 2026-04-17
Brand: `nanda AI`

## 1. Threadsで伸びやすい基本
Threadsは X よりも
- 共感
- 会話感
- 人っぽさ
- 温度
- 読みやすさ
が重要。

nanda AI では、Threadsは **少しやわらかく、でも実務示唆は残す** 方針でいく。

## 2. 基本構成

### パターンA, 共感から入る型
1. あるある
2. 気づき
3. 何が本質か
4. 自分の見解
5. やってみる価値

### パターンB, 会話型ノウハウ
1. 最近気づいたこと
2. 実はここが大事
3. こう考えると整理しやすい
4. すぐ使える視点

### パターンC, 軽いニュース再解釈型
1. 何があったか
2. 自分はこう見た
3. 日本だとこう使える
4. だから気にしたい

## 3. 書き方ルール
- Xより少し自然な話し言葉にする
- 断定より共感を混ぜる
- ただし中身は薄くしない
- ちょっと人柄が見える文にする
- 最後は会話が生まれる余白を残す

## 4. nanda AI向けのトーン
- 落ち着いている
- 親しみがある
- 情報に強い
- でも上から目線ではない
- ちゃんと考えている感じがある

## 5. 伸びやすい入り方
- 最近これをよく感じます
- これ、地味だけど大事です
- たぶんここを見落としやすい
- 正直、ここが分かれ目だと思っています
- 海外情報を見ていて思うのは

## 6. 避けること
- ニュース見出しの焼き直し
- 硬すぎる解説文
- いかにもAIっぽい無機質な文
- セールス感の強すぎるCTA

## 7. ゴール
- 反応しやすい
- 共感される
- 人柄が見える
- でも実務アカウントとして信用が落ちない
