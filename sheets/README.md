# Sheets

このディレクトリには、スプレッドシート運用の元になるテンプレを置く。

## 使い方
1. `x-thread-content-tracker-template.csv` をGoogleスプレッドシートへインポートする
2. 日本語列で使う場合は `x-thread-content-tracker-template-ja.csv` を使う
2. 列名は原則そのまま使う
3. 投稿ごとに1行で管理する
4. 投稿後は `posted_url` と `result_memo` を埋める

## 想定用途
- Xスレッド案の管理
- 承認ステータス管理
- 投稿日時の管理
- 投稿後の改善ログ管理

## テンプレ一覧
- `x-thread-content-tracker-template.csv` - 英字列の基本版
- `x-thread-content-tracker-template-ja.csv` - 現在の運用に合わせた日本語列版
