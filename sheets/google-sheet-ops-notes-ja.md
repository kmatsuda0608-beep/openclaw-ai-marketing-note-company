# Googleスプレッドシート運用メモ

## 現在の列構成
- `日付`
- `リサーチ情報`
- `X数値`
- `スレッズ数値`
- `X投稿文`
- `Threads投稿文`
- `共通ベーステキスト`
- `ハッシュタグ`
- `ステータス`
- `修正案`
- `AI更新`
- `X投稿URL`
- `Threads投稿URL`
- `振り返り分析`
- `振り返りFB`
- `再学習済み`

## この構成の意味
- `X数値` と `スレッズ数値` は、投稿後の実数値を入れていく列
- `X投稿文` と `Threads投稿文` は、各プラットフォーム向けに作った本文
- `共通ベーステキスト` は、両方に共通する核の文章
- `修正案` は人からの修正依頼
- `AI更新` は修正依頼を受けて私が再提案した本文
- `振り返り分析` は私の分析
- `振り返りFB` はおまつさんのフィードバック
- `再学習済み` は、そのFB内容を理解して今後の作成へ反映済みかどうか

## ステータスのおすすめ運用
- Draft
- Review
- Approved
- Posted
- Rework
- Hold

## 再学習済みの運用
- `未` : まだFBを反映していない
- `済` : FB内容を理解し、次回以降の作成に反映済み
- FBの意味が曖昧なら、先に質問してから `済` にする

## 運用メモ
- 投稿前は `X数値` と `スレッズ数値` を空欄でよい
- 投稿後に実数値を追記する
- 修正が入ったら `AI更新` に最新版を書く
- 伸びた / 伸びなかった理由は `振り返り分析` に残す
