# Samples

このディレクトリには、初回運用やレビュー用のサンプル成果物を置く。

## 含まれるもの
- `daily-materials-template.md` — 当日素材の入力テンプレ
- `daily-materials-sample-01.md` — 初回運用を想定した素材サンプル
- `approval-template.md` — 人レビュー用テンプレ
- `orchestrator/` — オーケストレーター出力サンプル
- `x-posts/` — X投稿サンプル
- `free-note/` — 無料noteサンプル
- `paid-note/` — 有料noteサンプル
- `funnels/` — 導線設計サンプル

## 使い方
1. `daily-materials-template.md` を複製して当日素材を作る
2. 各部隊へプロンプトを渡して草案を作る
3. サンプルと比較しながらレビュー観点を揃える
4. 採用 / 不採用をログへ残す
